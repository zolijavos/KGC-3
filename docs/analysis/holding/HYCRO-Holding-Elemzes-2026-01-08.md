## 1. Kontextus

### A Holding Koncepció

Egy új **holding cég** alapítása, amely összefogja:

| Entitás                 | Szerep                                                       |
| ----------------------- | ------------------------------------------------------------ |
| **KGC (Kisgépcentrum)** | Fizikai szolgáltatások: gépkölcsönzés, szerelés, értékesítés |
| **MyForge Labs**        | IT fejlesztés, szoftvermegoldások                            |
| **MyForge OS**          | Közös IT platform (ERP, eseményszervezés, SaaS megoldások)   |
