## 3. A Profit-Sharing Modell: ERP Fókuszú

### 3.1 Alapelv

**Csak az ERP üzletágra vonatkozik!** A MyForge OS platformon futó egyéb projektek (eseményszervező, weboldal builder, stb.) 100%-ban MyForge Labs tulajdona - ezek NEM osztódnak.

```
┌─────────────────────────────────────────────────────────────┐
│           ERP PROFIT-SHARING MODELL                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1️⃣ FRANCHISE ERP (KGC dominál)                            │
│     KGC viseli a munkát = 80/20 KGC/MFL                    │
│                                                             │
│  2️⃣ SAAS ERP (MFL dominál)                                 │
│     MFL végzi az értékesítést = 20/80 KGC/MFL              │
│                                                             │
│  3️⃣ TŐKEKOCKÁZAT (ROI First)                               │
│     Befektető kap 90%-ot megtérülésig                      │
│                                                             │
│  ⚠️  FONTOS: Csak ERP bevételek!                           │
│      Más SaaS termékek nem érintettek                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 Részletes ERP Profit-Sharing Táblázat

| Forgatókönyv                        | KGC | MFL | Partner | Megjegyzés                         |
| ----------------------------------- | --- | --- | ------- | ---------------------------------- |
| **KGC saját franchise (ROI fázis)** | 90% | 10% | -       | Amíg KGC befektetése nem térül meg |
| **KGC saját franchise (normál)**    | 80% | 20% | -       | Megtérülés után                    |
| **Partner franchise ERP**           | 25% | 15% | 60%     | Partner viseli a tőkekockázatot    |
| **SaaS ERP (MFL hozta ügyfelet)**   | 20% | 80% | -       | MFL végzi a munkát                 |
| **SaaS ERP (KGC hozta ügyfelet)**   | 30% | 70% | -       | +10% ügyfélszerzési bónusz         |
| **Vegyes/Közös ERP projekt**        | 50% | 50% | -       | Mindketten dolgoznak               |

### 3.3 Platform Díj Kezelése

Két lehetséges modell a MyForge OS platform használatára:

**Opció A: Beépített Platform Hozzájárulás**

- A fenti arányok már tartalmazzák a platform használat értékét
- MFL 20% a franchise-ból = részben platform kompenzáció
- Egyszerűbb elszámolás, kevesebb adminisztráció

**Opció B: Külön Platform Licenc**

```
Holding fizet MFL-nek:
├── Fix havi díj: X Ft/hó platform használatért
└── VAGY: Y Ft/aktív user/hó

A profit-sharing arányok ez után a maradék profitra vonatkoznak.
```

### 3.4 Mi Számít ERP Projektnek?

**A HOLDINGBA TARTOZIK (ERP):**

- KGC ERP franchise (fizikai boltok)
- KGC ERP SaaS (online előfizetés)
- KGC branded kiskereskedelmi szoftver
- Gépkölcsönző, szerviz modulok KGC ügyfeleknek

**NEM TARTOZIK A HOLDINGBA (MFL saját):**

- Eseményszervező alkalmazás
- Weboldal/landing page builder
- HR rendszer (nem KGC branded)
- Számlázó alkalmazás (nem KGC branded)
- Bármely más MyForge OS-re épülő termék

| ---------------------------- | --------------- |

### 3.7 Franchise Partner Kezelése

Ha NEM a KGC, hanem egy **külső franchise partner** fekteti be a tőkét:

```
ERP Bevétel elosztás háromfelé:

┌─────────────────────────────────────────────────────────────┐
│  Franchise Partner:  60%  (ő viseli a tőkekockázatot)      │
│  KGC (Holding):      25%  (brand, know-how, support)       │
│  MFL (Holding):      15%  (ERP fejlesztés, support)        │
└─────────────────────────────────────────────────────────────┘

A Holding 40%-os részének belső elosztása (25% + 15%):
→ KGC: 62.5%  |  MFL: 37.5%
```
